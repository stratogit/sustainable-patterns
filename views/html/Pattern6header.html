Pattern Description

Pattern Name: COMPUTER RE-USE
Context/Intent This pattern will showcase possibilities for extending the use of computers inside a company.
Problem After 2-3 computers are being replaced in despite that they are still useful, also cleaning data and configuration takes time.
Forces
  • Policies inside companies force changes after 2-3 years if use.
  • People demand changes due to mismanagement.
  • Cleaning computer takes time.
Solution
  • Reward employees who keep for longer time their tools
  • If computer returns computer employee is in charge of cleaning before handling
  • Hard drive from previous computer could be use in new one, so the hard drives are not destroyed and information stays with user.
  • When computer is replaced, donation to schools and libraries to avoid destruction.
Consequences
  • Users print less.
  • Increase time of use in computers inside an organization.
  • Less computers parts are wasted in replacement with new computers.
  • Users learn to be responsible with their offices tools.
Related Patterns GREEN PROCUREMENT, SOCIAL SHARING
